## 开发流程

```
Flutter >= 3.35.1
Dart >= 3.6.0
```

### 创建项目

```bash
npm install -g @maiguangyang/oxygen_cli
oxygen_cli create projectName

# 或
npx @maiguangyang/oxygen_cli create projectName
```

### 新建页面（三步）

1. 在 `lib/routing/route_path.dart` 定义路由路径：
```dart
static Route myPage = const Route(name: 'myPage', path: '/my-page');
```

2. 在 `lib/routing/routes.dart` 注册路由：
```dart
FlutterRouter(
  name: RoutePath.myPage.name,
  path: RoutePath.myPage.path,
  builder: (context, state) => const MyPageScreen(),
),
```

3. 在 `lib/features/my_page_screen/` 创建 Feature：
```
my_page_screen/
├── application/          # 业务逻辑（按需创建）
│   ├── notifiers/
│   └── providers/
├── data/                 # 数据层（按需创建）
│   └── index.dart
└── ui/                   # UI 层
    ├── desktop/index.dart
    ├── mobile/index.dart
    └── index.dart        # 平台判断入口
```

### 多客户端配置

1. 在 `lib/core/config/clients.dart` 新增/修改客户端配置：
```dart
class Clients {
  static const List<ClientItem> list = [
    ClientItem(
      clientName: '白牌',
      projectName: 'my_app',
      appName: 'My App',
      apiUrl: 'https://api.example.com',
      // ...
    ),
  ];
}
```

2. 运行 `dart ./scripts/build_cli.dart projectName="my_app"`

3. 重启 APP

### 多平台路由配置

路由在 `lib/routing/routes.dart` 中统一配置，通过 `FlutterRouter` 的属性控制行为：

```dart
FlutterRouter(
  name: RoutePath.home.name,
  path: RoutePath.home.path,
  builder: (context, state) => const HomeScreen(),
  auth: true,               // 需要登录
  isDefault: true,           // 默认首页
  width: 1200,               // 桌面端窗口宽度
  height: 800,               // 桌面端窗口高度
),
```

支持 ShellRoute（共享布局）：
```dart
FlutterRouter(
  isShell: true,
  appBar: AppBar(title: Text('标题')),
  bottomNavigationBar: BottomNavBar(),
  routes: [
    FlutterRouter(name: 'tab1', path: '/tab1', builder: ...),
    FlutterRouter(name: 'tab2', path: '/tab2', builder: ...),
  ],
),
```
